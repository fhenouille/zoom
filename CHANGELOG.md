# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère à [Semantic Versioning](https://semver.org/lang/fr/).

## [Non publié]

### Ajouté
- Structure complète du projet full-stack
- Frontend React 18 avec TypeScript et Vite
- Backend Spring Boot 3.x avec JPA
- Système de routing avec React Router v6
- State management avec TanStack Query
- Interface utilisateur avec Ant Design
- API REST pour la gestion des réunions
- Base de données H2 en mémoire
- Tests unitaires (JUnit et React Testing Library)
- Configuration Docker et Docker Compose
- Documentation complète (README, GETTING_STARTED)
- Configuration VS Code avec tasks et debugger
- Support GitHub Copilot avec prompts personnalisés

## [0.0.1] - 2025-11-13

### Ajouté
- Initialisation du projet
- Configuration de base du frontend et backend
- Première version de l'entité Meeting
- Endpoints API basiques (GET /api/meetings)
- Pages Home et Meetings
- Composants Layout, Header, Footer
